# telegrambotapi

## Telegram Bot API
for use this, just run command:

```bash
nano run_telegram_bot_api.sh   and replace your api keys

bash run_telegram_bot_api.sh &
```

or

```python
nano run_telegram_bot_api.py   and replace your api keys

python3 run_telegram_bot_api.sh &
```
